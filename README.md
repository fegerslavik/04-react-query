# TMDB Movie Search

Застосунок для пошуку фільмів через TMDB API, побудований на React, TypeScript, Vite та Axios.

## Запуск

1. Встановіть залежності:

```bash
npm install
```

2. Створіть `.env` на основі `.env.example` і додайте свій TMDB токен:

```env
VITE_TMDB_TOKEN=your_tmdb_token
```

3. Запустіть dev-сервер:

```bash
npm run dev
```

## Скрипти

- `npm run dev` - локальна розробка
- `npm run build` - production збірка
- `npm run lint` - перевірка ESLint
- `npm run format` - форматування Prettier

## Технології

- React 19
- TypeScript
- Axios
- CSS Modules
- modern-normalize
